# Simple State Machine

## Description

Simple finite state machine

## Installation

```bash
# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

## run unit tests in watch mode
$ npm run test:watch

# test coverage
$ npm run test:cov
```

## Building the app

```bash
# production build
$ npm run build
```

## Utility scripts

```bash
# formatting files
$ npm run format

# check lint
$ npm run lint
```
